This application is part of the final project for the Hybrid Web Development course from AMIKOM Yogyakarta. an HTTP REST API providing information about countries worldwide and displaying data on the world's oceans from a JSON file.

Countries are arranged alphabetically, allowing users to search for a specific country or filter countries by world region. Selecting a country from the list reveals more detailed information, including a map link and an option to add the country to favourites.
